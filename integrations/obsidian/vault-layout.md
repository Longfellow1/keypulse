# Vault Layout

## Core folders

- `Daily/` - daily rollups
- `Events/` - atomic evidence cards
- `Topics/` - durable themes and methods
- `Inbox/` - raw imports that still need review
- `Sources/` - clipped external references
- `Archive/` - retired material

## Card priorities

1. Thinking
2. Theme
3. Event

Time is an indexing aid, not the organizing principle.
